# render
Software renderer for 3D perspective projection graphics.

https://github.com/user-attachments/assets/a1bcaee3-b2e5-48cb-88c7-99f0c7e21fe1

# Features

- render arbitrary objects
- camera (mouse + wasd)
- back face culling
- z-buffer depth test
- flat lambert shading
- platform independant through dependency inversion
- CPU only: No GPU / OpenGL / DirectX whatsoever!
